Introduction

Java is a high-level, object-oriented programming language that is widely used for building applications.Java contains a standard library of objects, such as Array, Date, and Math, and a core set of language elements such as operators, control structures, and statements. Core Java can be extended for a variety of purposes by supplementing it with additional objects.

Syntax

Java uses a class-based structure and requires every application to have a main method.

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

Variables

Java is a statically typed language, meaning variables must have a declared type.

int age = 25;
double height = 5.9;
String name = "John";

Data Types

Java supports primitive data types such as int, float, boolean, and char.

boolean isJavaFun = true;
char grade = 'A';
float pi = 3.14f;

Methods

Methods in Java are defined inside classes and follow a specific syntax.

public class Main {
    static void greet(String name) {
        System.out.println("Hello, " + name);
    }
}

Object-Oriented Programming

Java is fully object-oriented and uses classes and objects.

class Dog {
    String name;
    Dog(String name) {
        this.name = name;
    }
    void bark() {
        System.out.println("Woof!");
    }
}
